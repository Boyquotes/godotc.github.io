2. 类型转换
| func                    | Description                                                                                    |
| ----------------------- | ---------------------------------------------------------------------------------------------- |
| `reinterpret_cast<T>()` | 强制类型转换,但是没有进行二进制转换(用于指针，引用，指针和位数相同的int，整数到指针之间的转换) | 
| `static_cast<T>()` | 转换类型，在编译期间强制转换，c隐式转换的替代|
| `dynamic_cast<T>()` |转换指针和引用, 会先检查指针指向的类型是否符合转换兼容，不符合会获得空指针|
|`const_cast<T>()`|修改 const 类型变量|
