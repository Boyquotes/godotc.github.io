# Cursor Operation

| 符号 | 跳转到         |
| :---: | :----: |
| `0`  | 行首           |
| `^`  | 第一个非空字符 |
| `$`     |    行尾|


|      符号       |                 跳转到                 |
|:---------------:|:--------------------------------------:|
|       `%`       |              配对的括号              |
|    `{` / `}`    |          上一段/下一段的开头           |
|    `(` / `)`    |           这一句/下一句开头            |
| ``[[`` / ``]]`` | 上一个/下一个函数 （要求 `{` 为单独行) |
|      `C-]`      |            函数或变量定义处            |
|  `C-t` /`C-0`   | 返回（函数) 跳转前位置                                       |
